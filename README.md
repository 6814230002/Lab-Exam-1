# Lab-Exam-1
Lab Exam 1
หัวข้อ: แกลเลอรีรูปภาพค้นหาได้ (Searchable Image Gallery)
• รายละเอียดโจทย์: ให้พัฒนาเว็บเพจที่มีช่องค้นหาและปุ่มสําหรับดึงรูปภาพจาก API รูปภาพ
เช่น Unsplash API หรือ Pixabay API เมื่อผู้ใช้ป้อนคําค้น (เช่น “แมว” หรือ “ทะเล”) แล้วกด
ค้นหา ให้ใช้ Fetch API ดึงรูปภาพที่เกี่ยวข้องจํานวน 6 ภาพ จากนั้นนํารูปเหล่านั้นมาแสดง
เป็นแกลเลอรีบนหน้าเว็บ
• สิ่งที่ต้องการให้แสดงผล: หลังผู้ใช้ค้นหา ให้แสดงภาพ thumbnail ขนาดย่อของผลลัพธ์ 6 รูป
จัดเรียงในรูปแบบกริด (เช่น 3x2) แต่ละรูปควรมีคําบรรยายใต้ภาพหรือข้อมูลผู้ถ่าย (ถ้ามีมา
ในข้อมูล JSON) เมื่อคลิกที่รูปภาพให้สามารถเปิดภาพขนาดเต็มได้ (อาจเป็นการลิงก์ไปยัง
URL ของภาพความละเอียดสูง)
• ทักษะที่ต้องการทดสอบ: การใช้ Fetch API พร้อมแนวทางการใส่พารามิเตอร์การค้นหาใน
URL (query string) เพื่อดึงข้อมูลที่ตรงกับคําค้น, การจัดการ JSON (ที่มีโครงสร้างซับซ้อนขึ้น
เช่น อาร์เรย์ของผลลัพธ์รูปภาพแต่ละรายการ), การจัดการ DOM เพื่อสร้างองค์ประกอบ
<img> หลายๆ ตัวตามจํานวนรูปที่ได้จากผลลัพธ์ และจัดวางด้วย CSS, การจัดรูปแบบข้อมูล
เพิ่มเติม (เช่น ถ้าจะใส่ชื่อช่างภาพใต้รูป), การใช้ Event ในการจับการ submit ของฟอร์ม
ค้นหา หรือการคลิกที่ปุ่มค้น

มีเปรี่ยบนจากโจทคืิอให้เป็นค้นหาแต่ API ที่หาได้มีแค่ของหมาอย่างเดียวเลยเปรี่ยนเป็นการสุ่มรูปแทน
